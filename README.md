그에 따른 반려동물 시장도 계속 커지고있습니다.</p>

<p>매번 사놓으면 반려동물이 쓰지 않아서 버리는 물건이 많았습니다.

무료 나눔하는 서비스를 기획하게 되었습니다 ❤</p>
